The Brautigam laboratory's equation for fitting protein thermal denaturation data is: m1T+b1+(m2T+b2)eHRT(1TTM)1+eHRT(1TTM)m1T+b1+(m2T+b2)eHRT(1TTM)1+eHRT(1TTM)
in which:

Symbol Meaning

| TT | absolute temperature, the independent variable | | arbitrary temperature-dependent signal reporting the denaturation KeqKeq | | m1m1 | slope of line in low-temperature region | | b1b1 | intercept of line in low-temperature region | | m2m2 | slope of line in high-temperature region | | b2b2 | intercept of line in high-temperature region | | HH | denaturation enthalpy, units as per choice of RR | | RR | the gas constant, in whichever units are preferred | | TMTM | melting temperature, Kelvin |

This equation is an adaptation of the integrated form of the van't Hoff equation, and my own derivation of it may be found here. The derivation clarifies that a a four-parameter result is 'on the way' to the six-parameter result above. In the notebook, I derive both, use each to treat a real data set, and apply an F-test to decide which provides the better fit.

: A sigmoidal equation of the form:f(x)=a+bex1+exf(x)=a+bex1+ex of which the fitting equation is an elaboration.

Encode the fitting function:

In [3]:
# if the value of a variable is specified in the definition of a function,
# that is its default value, and the function may be called without specifying 
# that parameter or its value. if, when called, the value of the variable
# is specified, that value overrides the default

# define the function
# note then that the default units for deltaH are J/mol
def tdmodel(T, m1, b1, m2, b2, deltaH, TM, R=8.314):
    line1=m1*T+b1
    line2=m2*T+b2
    Q=np.exp(-deltaH/(R*T)*(1-T/TM))
    return (line1+line2*Q)/(1+Q)

Good initial estimates for fittable parameters

Evidently this equation, which has 6 fittable parameters, is quite a bit more complicated than those necessary to fit linear and exponential decay data. With this number of variables, it's essential to have excellent initial parameter estimates. I walk through one example of how I accomplished that for a PupB NTSD mutant using actual Colbert Lab data.
